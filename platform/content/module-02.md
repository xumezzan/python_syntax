### Урок 2.1 — Коробка с наклейкой
**Экраны:** теория → допиши → задача

#### 📖 Теория
Переменная — это коробка с наклейкой-именем, в которую можно положить значение:

```python
name = "Пай"
print(name)
```
```
Пай
```

Знак `=` здесь не «равно», а «положи»: возьми то, что справа, и сохрани под именем слева.

Заметь: `print(name)` — без кавычек! С кавычками вывелось бы слово `name`, а без — содержимое коробки.

#### ✏️ Допиши код
**Условие:** сохрани в переменную `city` название города `Москва` и выведи его.
**Стартовый код:**
```python
city = ___
print(city)
```
**Подсказки:**
1. В коробку кладём текст — а текст всегда в кавычках.
2. Справа от `=` напиши `"Москва"`.
3. Решение: `city = "Москва"` — текст в кавычках, имя переменной без.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `Москва` |

#### 💻 Задача
**Условие:** создай переменную `food` со своим любимым блюдом и выведи её дважды (два print).
**Подсказки:**
1. Одна строка с `=`, две строки с `print`.
2. В обоих print — имя переменной без кавычек.
3. Решение: `food = "Суши"` затем `print(food)` два раза. В этом и сила переменной: написал один раз — используешь сколько хочешь.
**Автотесты:** вывод — две одинаковые непустые строки.

### Урок 2.2 — Имена переменных
**Экраны:** теория → квиз → баг

#### 📖 Теория
Имя переменной придумываешь ты, но есть правила:

- ✅ буквы, цифры, подчёркивание: `score`, `player_2`, `total_price`
- ❌ нельзя начинать с цифры: `2player`
- ❌ нельзя пробелы: `my score`
- ❌ нельзя имена команд: `print = 5` сломает print!

Договорённость программистов: слова разделяем подчёркиванием и пишем маленькими буквами — `best_friend`, а не `BestFriend`. Это называется **snake_case** — змеиный стиль. 🐍 Пай одобряет.

И важное: Python различает регистр — `name` и `Name` это две разные коробки.

#### ❓ Квиз
Какое имя переменной правильное?
- `user name` — ❌ пробел внутри имени запрещён.
- `1st_place` — ❌ имя не может начинаться с цифры.
- ✅ `first_place`
- `first-place` — ❌ дефис Python примет за минус.

#### 🐞 Найди баг
**Код:**
```python
my age = 25
print(my age)
```
**Ошибка:** строка 1 — пробел в имени переменной.
**Правильно:** `my_age = 25` и `print(my_age)`.
**Объяснение:** пробел разрезает имя надвое, Python видит два слова и не понимает их. Подчёркивание `_` — легальный «склеиватель» слов.

### Урок 2.3 — Числа и арифметика
**Экраны:** теория → допиши → задача

#### 📖 Теория
Числа пишутся без кавычек, и с ними Python умеет считать:

```python
apples = 3 + 4
print(apples)       # 7
print(10 - 2)       # 8
print(6 * 7)        # 42
print(15 / 3)       # 5.0
```

Целые числа называются `int` (от integer), дробные — `float` (плавающая точка). Дробная часть пишется через точку: `3.14`, не запятую!

Заметь: деление `/` всегда даёт float — даже `15 / 3` это `5.0`.

#### ✏️ Допиши код
**Условие:** посчитай, сколько минут в сутках. В сутках 24 часа, в часе 60 минут.
**Стартовый код:**
```python
minutes = 24 ___ 60
print(minutes)
```
**Подсказки:**
1. Часы и минуты надо перемножить.
2. Умножение в Python — звёздочка `*`.
3. Решение: `minutes = 24 * 60` → 1440.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `1440` |

#### 💻 Задача
**Условие:** ты купил 3 кофе по 250 рублей и булочку за 120. Сохрани сумму заказа в переменную `total` и выведи её.
**Подсказки:**
1. Сумма = кофе умножить на цену, плюс булочка.
2. `total = 3 * 250 + 120` — умножение выполнится раньше сложения, как в математике.
3. Решение: `total = 3 * 250 + 120` затем `print(total)` → 870.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `870` |

### Урок 2.4 — Целое деление, остаток, степень
**Экраны:** теория → квиз → задача

#### 📖 Теория
Кроме `+ - * /` есть ещё три оператора:

```python
print(17 // 5)   # 3  — целое деление: сколько раз влезло
print(17 % 5)    # 2  — остаток: сколько не влезло
print(2 ** 10)   # 1024 — степень: 2 в десятой
```

Представь 17 конфет на 5 друзей: каждому по `17 // 5 = 3` конфеты, и `17 % 5 = 2` останутся тебе. 😋

Остаток `%` — невероятно полезен: например, число чётное, если `x % 2` равно `0`.

#### ❓ Квиз
Что выведет `print(10 % 3)`?
- `3` — ❌ это результат `10 // 3` — целое деление.
- ✅ `1`
- `3.33` — ❌ это про обычное деление `/`.
- `0` — ❌ 10 не делится на 3 нацело, остаток есть.

#### 💻 Задача
**Условие:** в школе 134 ученика, в автобус помещается 30. Выведи двумя print: сколько полных автобусов поедет и сколько учеников останется ждать следующего.
**Подсказки:**
1. Полные автобусы — целое деление, оставшиеся — остаток.
2. `134 // 30` и `134 % 30`.
3. Решение: `print(134 // 30)` → 4, `print(134 % 30)` → 14.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `4` ⏎ `14` |

### Урок 2.5 — Переменная меняется
**Экраны:** теория → квиз → допиши

#### 📖 Теория
Содержимое коробки можно заменять. Самая важная строчка в программировании:

```python
score = 0
score = score + 10
print(score)   # 10
```

`score = score + 10` читается справа налево: «возьми текущее значение score, прибавь 10, положи обратно». Это настолько частая операция, что есть сокращение:

```python
score += 10   # то же, что score = score + 10
score -= 3    # то же, что score = score - 3
```

#### ❓ Квиз
Что выведет код?
```python
x = 5
x = x * 2
x += 1
print(x)
```
- `5` — ❌ значение x менялось дважды после присваивания.
- `10` — ❌ не забудь про `+= 1` на третьей строке.
- ✅ `11`
- `Ошибка` — ❌ переприсваивание — совершенно нормальная операция.

#### ✏️ Допиши код
**Условие:** к подписчикам блога прибавилось 250 человек. Обнови переменную сокращённой записью.
**Стартовый код:**
```python
followers = 1000
followers ___ 250
print(followers)
```
**Подсказки:**
1. Нужно «прибавь и сохрани» одним оператором.
2. Оператор из плюса и равно.
3. Решение: `followers += 250` → 1250.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `1250` |

### Урок 2.6 — Строки: склеивание и повторение
**Экраны:** теория → допиши → задача

#### 📖 Теория
Текст в Python называется строкой — `str` (от string). Строки можно складывать (склеивать) и умножать (повторять):

```python
first = "Пай"
greeting = "Привет, " + first + "!"
print(greeting)        # Привет, Пай!
print("ха" * 3)        # хахаха
```

Склеивание называют красивым словом **конкатенация**. Внимание на пробелы: `"Привет," + "Пай"` даст `Привет,Пай` — пробел сам не появится.

#### ✏️ Допиши код
**Условие:** склей имя и фамилию через пробел.
**Стартовый код:**
```python
first = "Ада"
last = "Лавлейс"
full = first + ___ + last
print(full)
```
**Подсказки:**
1. Между именем и фамилией должен оказаться пробел.
2. Пробел — это тоже строка: `" "`.
3. Решение: `full = first + " " + last` → `Ада Лавлейс`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `Ада Лавлейс` |

#### 💻 Задача
**Условие:** выведи строку аплодисментов: слово `хлоп-` повторённое 5 раз, одним print. Вывод: `хлоп-хлоп-хлоп-хлоп-хлоп-`
**Подсказки:**
1. Строку можно умножить на число.
2. `"хлоп-" * 5`.
3. Решение: `print("хлоп-" * 5)`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `хлоп-хлоп-хлоп-хлоп-хлоп-` |

### Урок 2.7 — Типы и type()
**Экраны:** теория → квиз → квиз

#### 📖 Теория
У каждого значения есть тип. Узнать его можно функцией `type()`:

```python
print(type(42))       # <class 'int'>   — целое
print(type(3.14))     # <class 'float'> — дробное
print(type("42"))     # <class 'str'>   — строка!
print(type(True))     # <class 'bool'>  — правда/ложь
```

Главная ловушка новичка: `42` и `"42"` — разные вещи! Первое — число, с ним можно считать. Второе — текст, который просто выглядит как число:

```python
print(42 + 1)        # 43
print("42" + "1")    # 421 — склеились как текст!
```

`True` и `False` — тип `bool`, «да» и «нет» мира программ. Он станет звездой модуля 3.

#### ❓ Квиз 1
Что выведет `print("5" + "5")`?
- `10` — ❌ это строки: для них `+` — склеивание, не сложение.
- ✅ `55`
- `Ошибка` — ❌ складывать строку со строкой можно.
- `"55"` — ❌ кавычки на экран не выводятся.

#### ❓ Квиз 2
Что выведет `print(5 + "5")`?
- `10` — ❌ нельзя сложить число и строку.
- `55` — ❌ для склеивания оба должны быть строками.
- ✅ `Ошибка: TypeError`
- `5 5` — ❌ так не бывает.

**Объяснение:** `TypeError: unsupported operand type(s) for +: 'int' and 'str'` — Python отказывается смешивать типы и честно говорит об этом. Как подружить число со строкой — следующий урок!

### Урок 2.8 — Преобразование типов
**Экраны:** теория → допиши → задача

#### 📖 Теория
Типы можно превращать друг в друга:

```python
print(int("42") + 1)     # 43  — строка стала числом
print(str(42) + "!")     # 42! — число стало строкой
print(float("3.5"))      # 3.5
print(int(7.9))          # 7   — int отбрасывает дробную часть (не округляет!)
```

`int("42")` сработает, только если в строке правда число: `int("привет")` уронит программу с `ValueError`.

#### ✏️ Допиши код
**Условие:** почини программу: сейчас она склеивает цены вместо сложения.
**Стартовый код:**
```python
price1 = "300"
price2 = "200"
total = ___(price1) + ___(price2)
print(total)
```
**Подсказки:**
1. Цены пришли строками — преврати их в числа.
2. Функция превращения в целое число — `int(...)`.
3. Решение: `total = int(price1) + int(price2)` → 500.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `500` |

#### 💻 Задача
**Условие:** переменная `level = 7`. Выведи строку `Уровень: 7` через склеивание `+` (не через запятую!).
**Подсказки:**
1. Склеивать можно только строку со строкой — а `level` число.
2. Преврати число в строку: `str(level)`.
3. Решение: `print("Уровень: " + str(level))`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `Уровень: 7` |

### Урок 2.9 — input: программа слушает
**Экраны:** теория → допиши → задача

#### 📖 Теория
До сих пор программы только говорили. Команда `input()` учит их слушать:

```python
name = input()
print("Привет,", name)
```

Программа остановится и будет ждать, пока пользователь что-нибудь напечатает и нажмёт Enter. Ответ ляжет в переменную.

**Суперважно:** `input()` ВСЕГДА возвращает строку. Даже если ввели `25` — в переменной будет `"25"`. Хочешь число — оборачивай: `age = int(input())`.

#### ✏️ Допиши код
**Условие:** программа спрашивает возраст и говорит, сколько будет через 10 лет. Почини пропуск.
**Стартовый код:**
```python
age = ___(input())
print(age + 10)
```
**Подсказки:**
1. `input` даёт строку, а нам нужно прибавить 10.
2. Оберни в функцию превращения в число.
3. Решение: `age = int(input())` — связка `int(input())` — самая частая пара в задачах.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `15` | `25` |
| `90` | `100` |

#### 💻 Задача
**Условие:** программа-эхо для попугая: прочитай слово и выведи его три раза через пробел. Ввод `кеша` → вывод `кеша кеша кеша`.
**Подсказки:**
1. Прочитай слово в переменную, выведи через запятые или склей.
2. `print(word, word, word)` — самый простой путь.
3. Решение: `word = input()` затем `print(word, word, word)`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `кеша` | `кеша кеша кеша` |
| `р-р-р` | `р-р-р р-р-р р-р-р` |

### Урок 2.10 — 🏆 БОСС: Калькулятор возраста
**Экраны:** босс (50 XP)

#### 📖 Вступление
🐍 Пай: «Соберём программу, которая знает о твоём возрасте больше, чем ты!»

#### 💻 Задание
**Условие:** программа читает две строки: имя и год рождения. Сейчас 2026 год. Вывод — ровно три строки:
```
Привет, Аня!
Тебе примерно 14 лет
Это около 5110 дней
```
Расчёт: возраст = 2026 − год рождения; дни = возраст × 365.

**Пример:**
| Ввод | Вывод |
|------|-------|
| `Аня` ⏎ `2012` | `Привет, Аня!` ⏎ `Тебе примерно 14 лет` ⏎ `Это около 5110 дней` |

**Подсказки:**
1. Два input: первый — строка-имя, второй — оберни в int. Потом две формулы и три print.
2. Каркас: `name = input()`, `year = int(input())`, `age = 2026 - year`, `days = age * 365`. Для вывода удобны запятые в print, но следи за `!` — её надо приклеить: `print("Привет, " + name + "!")`.
3. Решение:
```python
name = input()
year = int(input())
age = 2026 - year
days = age * 365
print("Привет, " + name + "!")
print("Тебе примерно", age, "лет")
print("Это около", days, "дней")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `Аня` ⏎ `2012` | `Привет, Аня!` ⏎ `Тебе примерно 14 лет` ⏎ `Это около 5110 дней` |
| `Макс` ⏎ `1990` | `Привет, Макс!` ⏎ `Тебе примерно 36 лет` ⏎ `Это около 13140 дней` |
| `Пай` ⏎ `2026` | `Привет, Пай!` ⏎ `Тебе примерно 0 лет` ⏎ `Это около 0 дней` |
