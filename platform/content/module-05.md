### Урок 5.1 — Буквы по номерам
**Экраны:** теория → квиз → допиши

#### 📖 Теория
Строка — это цепочка символов, и у каждого есть номер (индекс). Отсчёт — с нуля:

```
 п  и  т  о  н
 0  1  2  3  4
-5 -4 -3 -2 -1
```

```python
word = "питон"
print(word[0])    # п — первый символ
print(word[2])    # т
print(word[-1])   # н — минус значит «с конца»!
```

Отрицательные индексы — фишка Python: `[-1]` — последний символ, `[-2]` — предпоследний. Не нужно знать длину строки, чтобы взять её хвост.

Если попросить символ, которого нет (`word[99]`), — получишь `IndexError: string index out of range`.

#### ❓ Квиз
Что выведет код?
```python
s = "кода"
print(s[1])
```
- `к` — ❌ `к` стоит под номером 0 — отсчёт с нуля!
- ✅ `о`
- `од` — ❌ один индекс — один символ.
- `а` — ❌ `а` — это s[3] или s[-1].

#### ✏️ Допиши код
**Условие:** выведи первую и последнюю буквы слова — на одной строке через пробел.
**Стартовый код:**
```python
word = input()
print(word[___], word[___])
```
**Подсказки:**
1. Первая буква — нулевой индекс. Последняя — удобнее с конца.
2. `0` и `-1`.
3. Решение: `print(word[0], word[-1])` — работает для слова любой длины.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `питон` | `п н` |
| `ёж` | `ё ж` |

### Урок 5.2 — len: длина
**Экраны:** теория → квиз → задача

#### 📖 Теория
`len()` возвращает длину строки — количество символов:

```python
nick = "pyfan2026"
print(len(nick))        # 9
print(len(""))          # 0 — пустая строка
print(len("а б"))       # 3 — пробел тоже символ!
```

Связь индексов и длины: последний символ — это `s[len(s) - 1]`. Минус один — потому что считаем с нуля. Впрочем, ты уже знаешь способ короче: `s[-1]`.

#### ❓ Квиз
Что выведет `print(len("я ок"))`?
- `3` — ❌ пробел — полноценный символ, его тоже считаем.
- ✅ `4`
- `5` — ❌ кавычки не входят в строку.
- `Ошибка` — ❌ len работает с любой строкой.

#### 💻 Задача
**Условие:** Твиттер курильщика: прочитай сообщение. Если длина больше 20 символов — выведи `Слишком длинно`, иначе выведи `Опубликовано: N символов`, где N — длина.
**Подсказки:**
1. Сохрани `len(text)` в переменную и сравни с 20.
2. Во второй ветке выводи число через запятую в print.
3. Решение:
```python
text = input()
n = len(text)
if n > 20:
    print("Слишком длинно")
else:
    print("Опубликовано:", n, "символов")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `привет мир` | `Опубликовано: 10 символов` |
| `оченьдлинноесообщениеещёдлиннее` | `Слишком длинно` |

### Урок 5.3 — Срезы
**Экраны:** теория → допиши → задача

#### 📖 Теория
Срез вырезает кусок строки: `s[откуда:докуда]`. Правая граница, как в range, **не включается**:

```python
s = "программист"
print(s[0:8])    # программ
print(s[3:7])    # грам
print(s[:3])     # про  — пустое начало = «с начала»
print(s[8:])     # ист  — пустой конец = «до конца»
```

Мнемоника: срез `[a:b]` захватывает `b - a` символов. Исходная строка при этом не меняется — срез возвращает новую.

#### ✏️ Допиши код
**Условие:** из адреса `user@mail.ru` вырежи имя пользователя (до собачки) — `user`.
**Стартовый код:**
```python
email = "user@mail.ru"
print(email[___:___])
```
**Подсказки:**
1. Имя начинается с самого начала и длится 4 символа.
2. С нулевого по четвёртый не включая: `0:4` (или короче — `:4`).
3. Решение: `print(email[:4])` → `user`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `user` |

#### 💻 Задача
**Условие:** прочитай слово и выведи его без первой и последней буквы. Гарантируется, что в слове минимум 3 буквы. Ввод `котик` → `оти`.
**Подсказки:**
1. Начало среза — 1, конец — последний индекс, не включая.
2. Отрицательная граница работает и в срезах: `[1:-1]`.
3. Решение: `print(word[1:-1])`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `котик` | `оти` |
| `мир` | `и` |

### Урок 5.4 — Срез с шагом и разворот
**Экраны:** теория → квиз → задача

#### 📖 Теория
У среза есть третий параметр — шаг: `s[начало:конец:шаг]`:

```python
s = "абвгдеж"
print(s[::2])     # авде…  → абвгдеж через один: а в д ж
print(s[1::2])    # бгe…   → с первого через один
print(s[::-1])    # жедгвба — ВСЯ строка задом наперёд!
```

`[::-1]` — знаменитый трюк-разворот. Шаг −1 значит «иди справа налево». Однострочная проверка палиндрома: `s == s[::-1]`.

#### ❓ Квиз
Что выведет `print("123456"[::2])`?
- `246` — ❌ шаг 2 стартует с нулевого индекса — с единицы.
- ✅ `135`
- `123` — ❌ шаг 2 — это «через один», а не «первые три».
- `654321` — ❌ разворот — это шаг −1.

#### 💻 Задача
**Условие:** проверь, палиндром ли слово (читается одинаково в обе стороны). Выведи `Палиндром` или `Обычное слово`. Ввод `шалаш` → `Палиндром`.
**Подсказки:**
1. Сравни слово с его развёрнутой копией.
2. `word == word[::-1]`.
3. Решение:
```python
word = input()
if word == word[::-1]:
    print("Палиндром")
else:
    print("Обычное слово")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `шалаш` | `Палиндром` |
| `питон` | `Обычное слово` |
| `я` | `Палиндром` |

### Урок 5.5 — Методы: upper, lower, strip
**Экраны:** теория → допиши → задача

#### 📖 Теория
У строк есть встроенные «умения» — методы. Вызываются через точку:

```python
shout = "тише!"
print(shout.upper())     # ТИШЕ!
print("ЭЙ".lower())      # эй
print("  привет  ".strip())  # привет — пробелы по краям срезаны
```

Важно: методы не меняют строку, а возвращают новую. Хочешь сохранить результат — присвой:

```python
name = name.strip()
```

Классика: сравнение без учёта регистра — приведи обе строки к нижнему: `a.lower() == b.lower()`.

#### ✏️ Допиши код
**Условие:** пользователь ввёл ответ как попало: `  ДА `. Приведи к виду `да` — без пробелов и больших букв.
**Стартовый код:**
```python
answer = "  ДА "
clean = answer.___().___()
print(clean)
```
**Подсказки:**
1. Сначала срежь пробелы, потом опусти регистр (или наоборот — тут неважно).
2. `strip` и `lower`.
3. Решение: `clean = answer.strip().lower()` — методы можно вызывать цепочкой!
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `да` |

#### 💻 Задача
**Условие:** КАПС-ФИЛЬТР для чата. Прочитай сообщение. Если оно целиком в верхнем регистре (`msg == msg.upper()`) и длиннее 3 символов — выведи `Не кричи!`, иначе выведи сообщение как есть.
**Подсказки:**
1. Условие из двух частей через and.
2. `msg == msg.upper() and len(msg) > 3`.
3. Решение:
```python
msg = input()
if msg == msg.upper() and len(msg) > 3:
    print("Не кричи!")
else:
    print(msg)
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `ВСЕМ ПРИВЕТ` | `Не кричи!` |
| `всем привет` | `всем привет` |
| `ОК` | `ОК` |

### Урок 5.6 — replace, count, find
**Экраны:** теория → квиз → задача

#### 📖 Теория
Ещё три незаменимых метода:

```python
s = "кот любит котлеты"
print(s.replace("кот", "пёс"))  # пёс любит пёслеты — заменяет ВСЕ вхождения!
print(s.count("кот"))           # 2 — сколько раз встретилось
print(s.find("любит"))          # 4 — индекс, где начинается
print(s.find("заяц"))           # -1 — не нашёл
```

Заметь подвох в первой строке: `replace` заменил и «кот» внутри «котлеты». Методы не понимают смысла — только символы.

`find` возвращает `-1`, если не нашёл, — это удобно проверять через if.

#### ❓ Квиз
Что выведет `print("ха-ха-ха".replace("ха", "хи"))`?
- `хи-ха-ха` — ❌ replace заменяет все вхождения, не только первое.
- ✅ `хи-хи-хи`
- `ха-ха-ха` — ❌ replace вернул новую строку, мы её сразу печатаем.
- `Ошибка` — ❌ всё корректно.

#### 💻 Задача

**Подсказки:**
1. Один вызов replace решает всё.

3. Решение:

### Урок 5.7 — f-строки
**Экраны:** теория → допиши → задача

#### 📖 Теория
Помнишь мучения со склеиванием? `"Уровень: " + str(level)`… Забудь. Встречай **f-строки** — лучший способ собрать текст из кусочков:

```python
name = "Пай"
level = 7
print(f"Игрок {name}, уровень {level}")
```
```
Игрок Пай, уровень 7
```

Буква `f` перед кавычками, переменные — в фигурных скобках. Внутри скобок можно даже считать:

```python
print(f"Через год: {level + 1}")
```

Никаких str(), никаких плюсов и пробелов вручную. С этого урока используем f-строки везде.

**Стартовый код:**

**Подсказки:**
1. В фигурные скобки — имена переменных, без кавычек.
2. `{city}` и `{temp}`.
3. Решение: `print(f"В городе {city} сейчас {temp} градуса")`.
**Автотесты:**

### Урок 5.8 — startswith, endswith, isdigit
**Экраны:** теория → квиз → задача

#### 📖 Теория
Методы-проверки возвращают `True`/`False` — идеальны для if:

```python
file = "selfie.png"
print(file.endswith(".png"))     # True — это картинка
print(file.startswith("IMG"))    # False

code = "1234"
print(code.isdigit())            # True — только цифры
print("12a4".isdigit())          # False
```

`isdigit` — твой щит против падений `int(input())`: сначала проверь, потом превращай.

#### ❓ Квиз
Что выведет `print("python.pdf".endswith(".png"))`?
- `True` — ❌ файл кончается на `.pdf`, а спрашиваем про `.png`.
- ✅ `False`
- `Ошибка` — ❌ метод корректный.
- `.pdf` — ❌ endswith возвращает True/False, а не расширение.

#### 💻 Задача
**Условие:** валидатор номера. Прочитай строку. Если она начинается с `+7` и часть после `+7` состоит из 10 цифр — выведи `Номер принят`, иначе `Неверный формат`.
**Подсказки:**
1. Три проверки через and: startswith, длина хвоста, isdigit хвоста.
2. Хвост — это срез `phone[2:]`.
3. Решение:
```python
phone = input()
tail = phone[2:]
if phone.startswith("+7") and len(tail) == 10 and tail.isdigit():
    print("Номер принят")
else:
    print("Неверный формат")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `+79161234567` | `Номер принят` |
| `89161234567` | `Неверный формат` |
| `+7916123` | `Неверный формат` |
| `+7916123456a` | `Неверный формат` |

### Урок 5.10 — 🏆 БОСС: Шифр Цезаря
**Экраны:** теория-вставка → босс (50 XP)

#### 📖 Мини-теория
Юлий Цезарь шифровал письма, сдвигая каждую букву алфавита на 3 вперёд: А→Г, Б→Д… Алфавит при этом закольцован: Я→В.

План работы шифра на Python:
1. Заведи строку-алфавит: `abc = "абвгдежзийклмнопрстуфхцчшщъыьэюя"` (32 буквы, без ё).
2. Для каждой буквы слова найди её номер: `abc.find(ch)`.
3. Прибавь сдвиг и возьми остаток: `(номер + 3) % 32` — вот оно, закольцовывание через `%`!
4. Возьми букву с новым номером и приклей к результату.

#### 💻 Задание
**Условие:** прочитай слово (маленькие русские буквы, без ё) и выведи его, зашифрованное сдвигом +3.

**Пример:** `привет` → `тулеих`

**Подсказки:**
1. Это паттерн «накопитель» из модуля 4, только копишь не сумму, а строку: `result = ""`, в цикле `result += новая_буква`.
2. Внутри цикла: `i = abc.find(ch)`, потом `result += abc[(i + 3) % 32]`.
3. Решение:
```python
abc = "абвгдежзийклмнопрстуфхцчшщъыьэюя"
word = input()
result = ""
for ch in word:
    i = abc.find(ch)
    result += abc[(i + 3) % 32]
print(result)
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `привет` | `тулеих` |
| `яма` | `впг` — проверка закольцовки: я(31)→(31+3)%32=2→в |
| `абв` | `где` |

**Бонус-задание (⭐ +20 XP):** добавь дешифратор — второй строкой ввода приходит `+` (шифруем) или `-` (расшифровываем; сдвиг −3, в Python `(i - 3) % 32` тоже работает!).
