### Урок 10.1 — Читаем файл
**Экраны:** теория → допиши → квиз

#### 📖 Теория
Все твои программы страдали амнезией: перезапустил — всё забыто. Лекарство — файлы.

```python
f = open("diary.txt", encoding="utf-8")   # открыть
text = f.read()                            # прочитать ВСЁ в одну строку
f.close()                                  # закрыть!
print(text)
```

`open` открывает файл (нет такого — `FileNotFoundError`), `read` отдаёт содержимое строкой, `close` обязательно закрывает. `encoding="utf-8"` — чтобы русские буквы читались правильно, пиши его всегда.

Забыть close — как уйти из дома с открытой дверью. В следующем уроке Python сам будет закрывать за нами.

f = open("secret.txt", encoding="utf-8")

**Подсказки:**
1. Метод, читающий весь файл.
2. `read` — без аргументов.

### Урок 10.2 — with: файл закрывается сам
**Экраны:** теория → допиши

#### 📖 Теория
Профессиональный способ работы с файлами — блок `with`:

```python
with open("diary.txt", encoding="utf-8") as f:
    text = f.read()
print(text)
```

`with` гарантирует: файл закроется сам — при выходе из блока, даже если внутри случилась ошибка (привет, finally из модуля 9 — with делает это за тебя). С этого момента открываем файлы только так. Никаких close — Python сам.

#### ✏️ Допиши код
**Условие:** перепиши чтение на with.
**Предзагруженный файл note.txt:** `купить хлеб`
**Стартовый код:**
```python
___ open("note.txt", encoding="utf-8") ___ f:
    print(f.read())
```
**Подсказки:**
1. Конструкция из двух слов: «с файлом ... как f».
2. `with` и `as`.
3. Решение: `with open("note.txt", encoding="utf-8") as f:`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `купить хлеб` |

### Урок 10.3 — Файл по строчкам
**Экраны:** теория → задача

#### 📖 Теория
По файлу можно идти циклом — строка за строкой:

```python
with open("guests.txt", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

Подвох: каждая строка приходит с невидимым `\n` на хвосте. Не уберёшь — получишь двойные переносы. Поэтому `line.strip()` — рефлекторное движение при чтении строк.

Все паттерны циклов работают и здесь: счётчики, накопители, поиск максимума — только источник теперь файл.

#### 💻 Задача
**Условие:** в файле `guests.txt` — имена гостей, по одному на строку. Выведи `Гостей: N`.
**Предзагруженный файл guests.txt:**
```
Аня
Боря
Вика
Гоша
```
**Подсказки:**
1. Счётчик + цикл по файлу.
2. `count += 1` на каждую строку.
3. Решение:
```python
count = 0
with open("guests.txt", encoding="utf-8") as f:
    for line in f:
        count += 1
print(f"Гостей: {count}")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `Гостей: 4` |

### Урок 10.4 — Пишем в файл
**Экраны:** теория → допиши → задача

#### 📖 Теория
Второй аргумент open — режим:

```python
with open("log.txt", "w", encoding="utf-8") as f:   # w — write: с нуля
    f.write("Первая строка\n")
    f.write("Вторая строка\n")

with open("log.txt", "a", encoding="utf-8") as f:   # a — append: дописать
    f.write("Третья строка\n")
```

- `"w"` — записать заново. **Стирает всё, что было в файле!**
- `"a"` — дописать в конец, ничего не трогая.
- без режима — `"r"`, чтение.

И заметь: `write` не добавляет перенос строки сам — `\n` пишешь ты. Если файла не было — оба режима записи его создадут.

#### ✏️ Допиши код
**Условие:** допиши строку в дневник тренировок, НЕ стирая старые записи.
**Предзагруженный файл sport.txt:** `пн: бег\n`
**Стартовый код:**
```python
with open("sport.txt", ___, encoding="utf-8") as f:
    f.write("вт: турник\n")
```
**Подсказки:**
1. Режим «дописать», не «перезаписать».
2. Буква `a` в кавычках.
3. Решение: `open("sport.txt", "a", encoding="utf-8")`.
**Автотесты:** после запуска файл содержит `пн: бег` ⏎ `вт: турник`.

#### 💻 Задача
**Условие:** прочитай с клавиатуры 3 строки — план на день — и запиши их в файл `plan.txt`, каждую на своей строке. Потом (вторым with) прочитай файл и выведи содержимое — убедись, что записалось.
**Подсказки:**
1. Первый with в режиме "w" + цикл из трёх input. Второй — чтение и print.
2. Не забудь `+ "\n"` при записи каждой строки.
3. Решение:
```python
with open("plan.txt", "w", encoding="utf-8") as f:
    for i in range(3):
        f.write(input() + "\n")

with open("plan.txt", encoding="utf-8") as f:
    print(f.read().strip())
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `зал` ⏎ `код` ⏎ `сон` | `зал` ⏎ `код` ⏎ `сон` |

### Урок 10.5 — Обрабатываем данные из файла
**Экраны:** теория → задача

#### 📖 Теория
Настоящая сила — файл + split. Типичный формат «значения через разделитель»:

```
хлеб;40
молоко;90
```

```python
with open("prices.txt", encoding="utf-8") as f:
    for line in f:
        name, price = line.strip().split(";")
        print(f"{name} стоит {price}")
```

Строка → strip → split → распаковка. Эта четырёхходовка разбирает логи, выгрузки, CSV-таблицы — половину всех данных мира.

#### 💻 Задача
**Условие:** в файле `scores.txt` — результаты игроков в формате `имя;очки`. Выведи имя чемпиона (наибольшие очки; он один).
**Предзагруженный файл scores.txt:**
```
ника;340
тоха;520
муха;180
```
**Подсказки:**
1. Цикл по файлу + распаковка + поиск максимума (модуль 4!). Очки превращай в int.
2. Два накопителя: `best_name`, `best_score`.
3. Решение:
```python
best_name = ""
best_score = 0
with open("scores.txt", encoding="utf-8") as f:
    for line in f:
        name, score = line.strip().split(";")
        score = int(score)
        if score > best_score:
            best_score = score
            best_name = name
print(best_name)
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `тоха` |

### Урок 10.6 — Марафон: всё и сразу
**Экраны:** квиз → баг → задача

#### ❓ Квиз
В файле было `привет`. Что будет в файле после кода?
```python
with open("f.txt", "w", encoding="utf-8") as f:
    f.write("пока")
```
- `привет` ⏎ `пока` — ❌ режим "w" стирает старое содержимое.
- ✅ только `пока`
- `приветпока` — ❌ для дописывания нужен режим "a".
- Ошибка — ❌ перезапись — штатная операция.

#### 🐞 Найди баг
**Код:**
```python
with open("words.txt", encoding="utf-8") as f:
    for line in f:
        if line == "стоп":
            print("Нашёл!")
```
**Задумка:** найти в файле строку `стоп` (файл содержит её).
**Ошибка:** строка 3 — `line` приходит как `"стоп\n"`, сравнение всегда ложно.
**Правильно:** `if line.strip() == "стоп":`.
**Объяснение:** невидимый `\n` — причина №1 «почему не находится то, что точно есть».

#### 💻 Задача
**Условие:** подсчёт зарплаты курьера. В файле `delivery.txt` — суммы доставок, по одной на строку. Выведи две строки: `Доставок: N` и `Заработано: X` (X — сумма, курьер получает 10% от каждой доставки, бери `сумма // 10`).
**Предзагруженный файл delivery.txt:**
```
500
1200
800
```
**Подсказки:**
1. Два накопителя: счётчик и сумма.
2. `earned += int(line) // 10`.
3. Решение:
```python
count = 0
earned = 0
with open("delivery.txt", encoding="utf-8") as f:
    for line in f:
        count += 1
        earned += int(line.strip()) // 10
print(f"Доставок: {count}")
print(f"Заработано: {earned}")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `Доставок: 3` ⏎ `Заработано: 250` |

### Урок 10.7 — 🏆 ФИНАЛЬНЫЙ БОСС, часть 1: TO-DO ядро
**Экраны:** босс (50 XP)

#### 📖 Вступление
🐍 Пай: «Финал! Соберём настоящее приложение — список дел. Сегодня — ядро на функциях, завтра прикрутим память на файлах. В этом проекте есть ВСЁ, что ты выучил: списки, словари, циклы, условия, функции, try/except».

#### 💻 Задание
**Условие:** TO-DO без файлов. Дела храним списком словарей: `{"текст": ..., "сделано": True/False}`. Программа читает команды в цикле:
- `добавить <текст дела>` → добавляет (текст может быть из нескольких слов!), печатает `Добавлено`
- `список` → печатает дела: `1. [x] купить хлеб` (сделано) / `2. [ ] помыть кота` (нет); если дел нет — `Дел нет, отдыхай!`
- `сделал <номер>` → отмечает дело по номеру (с 1), печатает `Молодец!`; кривой номер → `Нет такого дела`
- `выход` → `Пока!` и конец

Обязательно: логика — в функциях `add(todos, text)`, `show(todos)`, `done(todos, num_str)`; разбор «команда + хвост» — через `split(" ", 1)` (режет только по первому пробелу!).

**Подсказки:**
1. `parts = line.split(" ", 1)` даёт `["добавить", "купить хлеб"]`. Команда — parts[0], хвост — parts[1].
2. В `done` оберни int() в try/except ValueError и проверь диапазон. Отметка: `todos[n - 1]["сделано"] = True`.
3. Решение:
```python
def add(todos, text):
    todos.append({"текст": text, "сделано": False})
    print("Добавлено")

def show(todos):
    if len(todos) == 0:
        print("Дел нет, отдыхай!")
        return
    n = 1
    for todo in todos:
        if todo["сделано"]:
            mark = "x"
        else:
            mark = " "
        print(f"{n}. [{mark}] {todo['текст']}")
        n += 1

def done(todos, num_str):
    try:
        n = int(num_str)
        if n < 1:
            print("Нет такого дела")
            return
        todos[n - 1]["сделано"] = True
        print("Молодец!")
    except ValueError:
        print("Нет такого дела")
    except IndexError:
        print("Нет такого дела")

todos = []
while True:
    parts = input().split(" ", 1)
    cmd = parts[0]
    if cmd == "выход":
        print("Пока!")
        break
    elif cmd == "добавить":
        add(todos, parts[1])
    elif cmd == "список":
        show(todos)
    elif cmd == "сделал":
        done(todos, parts[1])
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `список` ⏎ `выход` | `Дел нет, отдыхай!` ⏎ `Пока!` |
| `добавить купить хлеб` ⏎ `добавить помыть кота` ⏎ `сделал 1` ⏎ `список` ⏎ `выход` | `Добавлено` ⏎ `Добавлено` ⏎ `Молодец!` ⏎ `1. [x] купить хлеб` ⏎ `2. [ ] помыть кота` ⏎ `Пока!` |
| `добавить дело` ⏎ `сделал 5` ⏎ `сделал ага` ⏎ `выход` | `Добавлено` ⏎ `Нет такого дела` ⏎ `Нет такого дела` ⏎ `Пока!` |

### Урок 10.8 — 🏆 ФИНАЛЬНЫЙ БОСС, часть 2: память
**Экраны:** босс (50 XP)

#### 💻 Задание
**Условие:** добавь приложению память — два изменения:
1. Функция `save(todos)` — пишет дела в `todos.txt` в формате `1|купить хлеб` (первый символ: 1 — сделано, 0 — нет; разделитель `|`). Вызывай её после каждого изменения (добавить/сделал).
2. Функция `load()` — при старте читает `todos.txt`, если он есть, и возвращает список дел; файла нет (`FileNotFoundError`) — возвращает пустой список.

Поведение команд — как в части 1. Теперь после `выход` и нового запуска дела на месте!

**Подсказки:**
1. `save`: with open "w" + цикл + `f.write(f"{флаг}|{текст}\n")`. Флаг сделанности в 1/0: `if todo["сделано"]: flag = "1"`.
2. `load`: оберни ВЕСЬ with в `try/except FileNotFoundError: return []`. Разбор строки: `flag, text = line.strip().split("|", 1)`, сделано — это `flag == "1"`.
3. Решение (новые функции + изменения):
```python
def save(todos):
    with open("todos.txt", "w", encoding="utf-8") as f:
        for todo in todos:
            if todo["сделано"]:
                flag = "1"
            else:
                flag = "0"
            f.write(f"{flag}|{todo['текст']}\n")

def load():
    try:
        todos = []
        with open("todos.txt", encoding="utf-8") as f:
            for line in f:
                flag, text = line.strip().split("|", 1)
                todos.append({"текст": text, "сделано": flag == "1"})
        return todos
    except FileNotFoundError:
        return []

# в основной программе:
todos = load()
# ...и после каждого add(...) и done(...) — вызов save(todos)
```

| `добавить тест` ⏎ `выход` | файл todos.txt содержит `0\|тест` |
| предзагружен todos.txt: `1\|старое дело` → `список` ⏎ `выход` | `1. [x] старое дело` ⏎ `Пока!` |
| `добавить а` ⏎ `сделал 1` ⏎ `выход`, перезапуск, `список` ⏎ `выход` | `1. [x] а` во втором запуске |
