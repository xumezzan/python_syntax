### Урок 7.1 — Словарь: данные с именами
**Экраны:** теория → допиши → квиз

#### 📖 Теория
В списке отделения пронумерованы. В словаре (`dict`) — подписаны:

```python
hero = {"имя": "Пай", "уровень": 7, "класс": "змей-маг"}
print(hero["имя"])      # Пай
print(hero["уровень"])  # 7
```

Фигурные скобки, пары `ключ: значение` через запятую. Доступ — по ключу, а не по номеру.

Это естественный способ описать ОДНУ вещь с разными свойствами: игрока, товар, контакт. Спросишь несуществующий ключ — получишь `KeyError`.

#### ✏️ Допиши код
**Условие:** выведи цену пиццы из меню.
**Стартовый код:**
```python
menu = {"пицца": 599, "паста": 449, "суп": 299}
print(menu[___])
```
**Подсказки:**
1. Ключ — это строка, не забудь кавычки.
2. `"пицца"`.
3. Решение: `print(menu["пицца"])` → 599.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `599` |

#### ❓ Квиз
Что выведет код?
```python
d = {"a": 1, "b": 2}
print(d["b"])
```
- `1` — ❌ по ключу "b" лежит 2.
- ✅ `2`
- `b` — ❌ выводится значение, а не ключ.
- `Ошибка` — ❌ ключ "b" существует.

### Урок 7.2 — Добавить, изменить, удалить
**Экраны:** теория → допиши → задача

#### 📖 Теория
Словарь — живой: пары можно добавлять, менять и удалять:

```python
scores = {"Аня": 10}
scores["Боря"] = 7        # добавить новую пару
scores["Аня"] = 15        # изменить существующую
del scores["Боря"]        # удалить
print(scores)             # {'Аня': 15}
```

Одна и та же запись `d[ключ] = значение` и добавляет, и изменяет: есть ключ — перезапишет, нет — создаст. Пустой словарь — `{}`.

#### ✏️ Допиши код
**Условие:** добавь в профиль новое поле `город` со значением `Сочи`.
**Стартовый код:**
```python
profile = {"ник": "pyfan"}
___ = "Сочи"
print(profile)
```
**Подсказки:**
1. Та же запись, что и чтение, но слева от `=`.
2. `profile["город"]`.
3. Решение: `profile["город"] = "Сочи"`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `{'ник': 'pyfan', 'город': 'Сочи'}` |

#### 💻 Задача
**Условие:** банковский счёт: `account = {"баланс": 1000}` (задай в коде). Прочитай число — сумму покупки. Уменьши баланс на эту сумму и выведи `Остаток: X`.
**Подсказки:**

3. Решение:

(внутри f-строки с двойными кавычками ключ пишем в одинарных!)

### Урок 7.3 — in и get: безопасное чтение
**Экраны:** теория → квиз → задача

#### 📖 Теория
Спросить словарь о несуществующем ключе — упасть с `KeyError`. Два способа застраховаться:

```python
prices = {"кофе": 200}

# способ 1: проверить заранее
if "чай" in prices:
    print(prices["чай"])

# способ 2: get — вернёт None или запасное значение
print(prices.get("чай"))         # None
print(prices.get("чай", 0))      # 0 — «если нет, считай 0»
```

`in` для словаря проверяет КЛЮЧИ. `get` со вторым аргументом — элегантный способ задать «значение по умолчанию».

#### ❓ Квиз
Что выведет код?
```python
d = {"x": 5}
print(d.get("y", 100))
```
- `5` — ❌ спрашиваем ключ "y", не "x".
- ✅ `100`
- `None` — ❌ None был бы без второго аргумента.
- `Ошибка KeyError` — ❌ в этом и смысл get: он не падает.

#### 💻 Задача
**Условие:** переводчик с кошачьего: `{"мяу": "привет", "мур": "люблю", "шшш": "уйди"}` (задай в коде). Прочитай слово. Выведи перевод, а если слова нет в словаре — `Не понимаю`.
**Подсказки:**
1. get со вторым аргументом — задача в одну строку после словаря.
2. `cat.get(word, "Не понимаю")`.
3. Решение:
```python
cat = {"мяу": "привет", "мур": "люблю", "шшш": "уйди"}
word = input()
print(cat.get(word, "Не понимаю"))
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `мяу` | `привет` |
| `гав` | `Не понимаю` |

### Урок 7.4 — Цикл по словарю
**Экраны:** теория → допиши → задача

#### 📖 Теория
По словарю можно ходить циклом:

```python
prices = {"кофе": 200, "чай": 100}

for key in prices:               # по ключам
    print(key)

for key, value in prices.items():   # по парам — самый полезный способ!
    print(f"{key} стоит {value}")
```
```
кофе стоит 200
чай стоит 100
```

`.items()` выдаёт пары, которые тут же распаковываются в две переменные — как кортежи из модуля 6. Есть ещё `.keys()` и `.values()` — только ключи и только значения (например, `sum(prices.values())`).

#### ✏️ Допиши код
**Условие:** выведи табель: каждая строка — `предмет: оценка`.
**Стартовый код:**
```python
report = {"математика": 5, "история": 4}
for subject, grade in report.___():
    print(f"{subject}: {grade}")
```
**Подсказки:**
1. Метод, выдающий пары «ключ-значение».
2. `items`.
3. Решение: `report.items()`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `математика: 5` ⏎ `история: 4` |

#### 💻 Задача
**Условие:** карманные деньги: `spent = {"еда": 500, "кино": 400, "книги": 300}` (задай в коде). Выведи две строки: `Всего: 1200` и категорию с самой большой тратой: `Больше всего: еда`.
**Подсказки:**
1. Сумма — `sum(spent.values())`. Максимум ищи циклом по items с накопителем.
2. Храни два накопителя: `best_cat` и `best_val`, обновляй по if.
3. Решение:
```python
spent = {"еда": 500, "кино": 400, "книги": 300}
print(f"Всего: {sum(spent.values())}")
best_cat = ""
best_val = 0
for cat, val in spent.items():
    if val > best_val:
        best_val = val
        best_cat = cat
print(f"Больше всего: {best_cat}")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `Всего: 1200` ⏎ `Больше всего: еда` |

### Урок 7.5 — Словарь-счётчик
**Экраны:** теория → задача

#### 📖 Теория
Суперпаттерн: посчитать, чего сколько. Например, голоса в чате:

```python
votes = ["кот", "пёс", "кот", "кот"]
count = {}
for v in votes:
    if v in count:
        count[v] += 1     # уже встречали — плюс один
    else:
        count[v] = 1      # первый раз — заводим счёт
print(count)              # {'кот': 3, 'пёс': 1}
```

Или короче, через знакомый `get`:

```python
count[v] = count.get(v, 0) + 1
```

Этим паттерном считают слова в текстах, товары в корзинах, лайки, баги… Запомни его тело наизусть.

#### 💻 Задача
**Условие:** прочитай строку и посчитай каждую букву (пробелы пропусти). Выведи словарь. Ввод `мама` → `{'м': 2, 'а': 2}`.
**Подсказки:**
1. Цикл по строке + словарь-счётчик + if для пробела.
2. `if ch == " ": continue`.
3. Решение:
```python
text = input()
count = {}
for ch in text:
    if ch == " ":
        continue
    count[ch] = count.get(ch, 0) + 1
print(count)
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `мама` | `{'м': 2, 'а': 2}` |
| `о о о` | `{'о': 3}` |

### Урок 7.6 — Список словарей
**Экраны:** теория → квиз → задача

#### 📖 Теория
Структуры можно вкладывать друг в друга. Список словарей — так выглядят почти все данные в реальных программах:

```python
playlist = [
    {"название": "Кукла колдуна", "минуты": 4},
    {"название": "Группа крови", "минуты": 5},
]
print(playlist[0]["название"])   # Кукла колдуна
for song in playlist:
    print(song["минуты"])
```

Читай `playlist[0]["название"]` слева направо: возьми нулевой элемент списка (это словарь), у него — значение по ключу. Именно в таком виде приходят ответы от любых API — будешь работать с интернетом, увидишь это повсюду.

#### ❓ Квиз
Как достать `почту` второго пользователя?
```python
users = [
    {"ник": "ana", "почта": "a@x.ru"},
    {"ник": "bob", "почта": "b@x.ru"},
]
```
- `users["почта"][1]` — ❌ users — список, у него числовые индексы.
- ✅ `users[1]["почта"]`
- `users[2]["почта"]` — ❌ второй пользователь — индекс 1, считаем с нуля!
- `users[1].почта` — ❌ через точку — методы; значения словаря — в скобках.

#### 💻 Задача
**Условие:** задай в коде список товаров:
```python
goods = [
    {"имя": "мышь", "цена": 1500, "в_наличии": True},
    {"имя": "клавиатура", "цена": 4500, "в_наличии": False},
    {"имя": "коврик", "цена": 800, "в_наличии": True},
]
```
Выведи имена только тех товаров, что в наличии, — по одному на строку.
**Подсказки:**
1. Цикл по списку + if по ключу `в_наличии`.
2. `if item["в_наличии"]:` — bool сравнивать с True не нужно.
3. Решение:
```python
for item in goods:
    if item["в_наличии"]:
        print(item["имя"])
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| — | `мышь` ⏎ `коврик` |

### Урок 7.7 — Множества
**Экраны:** теория → квиз → допиши

#### 📖 Теория
Множество (`set`) — коллекция без повторов и без порядка:

```python
tags = {"питон", "код", "питон", "код"}
print(len(tags))            # 2 — дубли исчезли сами!

nums = [1, 2, 2, 3, 3, 3]
unique = set(nums)          # {1, 2, 3} — мгновенная чистка дублей
```

Проверка `in` у множества молниеносна даже на миллионе элементов — это его суперсила. Добавить — `add`, удалить — `discard`.

Внимание: `{}` — это пустой СЛОВАРЬ. Пустое множество — `set()`.

#### ❓ Квиз
Что выведет `print(len(set("шалаш")))`?
- `5` — ❌ set оставит каждую букву по одному разу.
- ✅ `3`
- `4` — ❌ буквы «ш» две и «а» две: остаются ш, а, л.
- `Ошибка` — ❌ set строки — множество её символов.

#### ✏️ Допиши код
**Условие:** сколько разных слов в предложении?
**Стартовый код:**
```python
words = input().split()
unique = ___(words)
print(len(unique))
```
**Подсказки:**
1. Преврати список в коллекцию без повторов.
2. Функция `set()`.
3. Решение: `unique = set(words)`.
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `да нет да может да` | `3` |
| `раз два три` | `3` |

### Урок 7.8 — Операции множеств
**Экраны:** теория → задача

#### 📖 Теория
Множества умеют сравниваться друг с другом — как круги на диаграмме:

```python
my_friends = {"Аня", "Боря", "Вика"}
your_friends = {"Вика", "Гоша"}

print(my_friends & your_friends)   # {'Вика'} — пересечение: общие
print(my_friends | your_friends)   # все вместе — объединение
print(my_friends - your_friends)   # {'Аня', 'Боря'} — только мои
```

`&` — «и там и там», `|` — «хотя бы где-то», `-` — «в первом, но не во втором». Одной операцией — то, что циклами писалось бы в пять строк.

#### 💻 Задача
**Условие:** два киномана вводят любимые фильмы: каждый — одна строка, фильмы через пробел (названия из одного слова). Выведи количество общих фильмов и сами фильмы отсортированным списком. Формат: первая строка `Общих: N`, вторая — отсортированный список.
**Подсказки:**
1. Два input().split(), оба в set, потом `&`.
2. Для красивого вывода: `sorted(common)` — даёт отсортированный список.
3. Решение:
```python
a = set(input().split())
b = set(input().split())
common = a & b
print(f"Общих: {len(common)}")
print(sorted(common))
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `Шрек Барби Дюна` ⏎ `Дюна Оно Шрек` | `Общих: 2` ⏎ `['Дюна', 'Шрек']` |
| `Кот` ⏎ `Пёс` | `Общих: 0` ⏎ `[]` |

### Урок 7.9 — Марафон повторения
**Экраны:** квиз → баг → задача

#### ❓ Квиз
Что выведет код?
```python
d = {}
d["a"] = 1
d["a"] = d["a"] + 10
print(d)
```
- `{'a': 1}` — ❌ вторая строка перезаписала значение.
- ✅ `{'a': 11}`
- `{'a': 1, 'a': 11}` — ❌ ключи в словаре не повторяются: запись по ключу заменяет.
- `Ошибка` — ❌ читать только что записанный ключ можно.

#### 🐞 Найди баг
**Код:**
```python
ages = {"Тим": 12, "Лия": 14}
print(ages[Тим])
```
**Ошибка:** строка 2 — ключ без кавычек; Python ищет переменную `Тим` → `NameError`.
**Правильно:** `print(ages["Тим"])`.

#### 💻 Задача
**Условие:** голосование за столицу мемов. Читай города, пока не введут `стоп`. Выведи город-победитель (он один, гарантируется). 
**Подсказки:**
1. Словарь-счётчик + поиск максимума — оба паттерна этого модуля.
2. Сначала собери `count`, потом пройди items и найди лучший.
3. Решение:
```python
count = {}
while True:
    city = input()
    if city == "стоп":
        break
    count[city] = count.get(city, 0) + 1
best = ""
best_n = 0
for city, n in count.items():
    if n > best_n:
        best_n = n
        best = city
print(best)
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `Омск` ⏎ `Питер` ⏎ `Омск` ⏎ `стоп` | `Омск` |
| `Тверь` ⏎ `стоп` | `Тверь` |

### Урок 7.10 — 🏆 БОСС: Телефонная книга
**Экраны:** босс (50 XP)

#### 📖 Вступление
🐍 Пай: «Соберём настоящее приложение с командами — как у взрослых. Словарь — идеальное хранилище контактов».

#### 💻 Задание
**Условие:** программа в цикле читает команды:
- `добавить <имя> <номер>` → сохраняет контакт, печатает `Сохранено`
- `найти <имя>` → печатает номер или `Нет такого контакта`
- `удалить <имя>` → удаляет и печатает `Удалено` (или `Нет такого контакта`)
- `все` → печатает контакты в формате `имя: номер`, отсортированные по имени
- `выход` → печатает `До связи!` и завершается

**Подсказки:**
1. Каркас: `while True` + `parts = input().split()` + цепочка if/elif по `parts[0]`. Контакты — словарь имя→номер.
2. Для «все»: `for name in sorted(book):` — sorted по словарю даёт отсортированные ключи. Для «найти/удалить» сначала проверь `in`.
3. Решение:
```python
book = {}
while True:
    parts = input().split()
    cmd = parts[0]
    if cmd == "выход":
        print("До связи!")
        break
    elif cmd == "добавить":
        book[parts[1]] = parts[2]
        print("Сохранено")
    elif cmd == "найти":
        if parts[1] in book:
            print(book[parts[1]])
        else:
            print("Нет такого контакта")
    elif cmd == "удалить":
        if parts[1] in book:
            del book[parts[1]]
            print("Удалено")
        else:
            print("Нет такого контакта")
    elif cmd == "все":
        for name in sorted(book):
            print(f"{name}: {book[name]}")
```
**Автотесты:**
| Ввод | Ожидаемый вывод |
|------|------------------|
| `добавить мама 112233` ⏎ `найти мама` ⏎ `выход` | `Сохранено` ⏎ `112233` ⏎ `До связи!` |
| `найти ктото` ⏎ `выход` | `Нет такого контакта` ⏎ `До связи!` |
| `добавить боб 1` ⏎ `добавить аня 2` ⏎ `все` ⏎ `удалить боб` ⏎ `все` ⏎ `выход` | `Сохранено` ⏎ `Сохранено` ⏎ `аня: 2` ⏎ `боб: 1` ⏎ `Удалено` ⏎ `аня: 2` ⏎ `До связи!` |
